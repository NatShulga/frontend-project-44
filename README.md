Description.

This project created 5 console games:

    ~ Game: "Parity check"

    ~ Game: "Calculator"

    ~ Game "GCD"

    ~ Game "Arithmetic progression"

    ~ Game "Is the number prime?"

Games launch commands:

    ~ brain-even

    ~ brain-calc

    ~ brain-gcd

    ~ brain-prime

    ~ brain-progression
